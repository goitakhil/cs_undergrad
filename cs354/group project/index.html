Rust is a systems programming language which usually refers to a language
used for developing system software as opposed to application software.
It has also been referred to as a general purpose and multi paradigm language.

The language was originally just a pet project by Mozilla employee Grayden Hoare.
In 2009 Mozilla began sponsoring the language. They announced it's development
in 2010 and it's first stable release was in 2015.

Rust's goal is to create highly concurrent, highly safe systems.
It does not permit segfaults, null pointers, or dangling pointers.
It does not use automated garbage collection. It is not object oriented,
but like C, can be implemented in an OO manner.

It's emphasis in high concurrency means that as the program can be decomposed into different units,
those concurrent units can be executed out of order and still arrive at the same result. This means that
these concurrent units can be executed parallely which significantly increases the speed of execution.
