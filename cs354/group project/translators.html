There is an interpreter and build simulator available here

You can download the compiler from here

On Linux and Mac you can install the compiler by running:
curl -sSf https://static.rust-lang.org/rustup.sh | sh

To compile a rust file called main.rs:
rustc main.rs

To run the output program:
./main

Most rust projects utilize a build system called Cargo, which is built for rust. It handles the debug and release build settings. It accepts commands similar to those used by make.

cargo build             // builds the main program and all dependencies 
cargo run               // builds the program and runs the main program
cargo build --release   // builds with extra optimizations and no debug info
cargo package           // packs the project into a source tarball
cargo test              // builds if needed, then runs a test suite (written by user)
cargo clean             // removes binaries and intermediate files
